# Zenith Components

Reusable section snippets live in `src/components/sections/`.

Use these as modular building blocks for new pages and niche variants.

## Included Sections

- `hero.html`
- `problem-solution.html`
- `features.html`
- `social-proof.html`
- `pricing.html`
- `faq.html`

`hero.html`, `pricing.html`, and `faq.html` are structured as standalone copy-paste snippets for rapid assembly.

## Usage

1. Copy the section snippet into a new page.
2. Replace placeholder copy using `docs/copywriting-cheatsheet.md`.
3. Replace placeholder images with optimized assets before release.
4. Keep class names unchanged to inherit Zenith styles.
