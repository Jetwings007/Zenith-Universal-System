# Zenith Copywriting Cheat Sheet

Use these formulas to write faster and improve conversion quality.

## Hero

- **Headline Formula**
  `[Outcome] for [Audience] without [Pain]`
- **Subheadline Formula**
  `Zenith helps [Audience] achieve [Outcome] in [Timeframe] with [Unique Mechanism].`
- **CTA Formula**
  `Start [Desired Outcome] today`

## Problem -> Solution

- **Problem Line**
  `Most [Audience] struggle with [Pain] because [Cause].`
- **Solution Line**
  `Zenith solves this with [System], so you can [Outcome].`

## Features

For each feature:

- **Feature Name**
- **Benefit Statement**
  `So you can [Practical Win].`
- **Proof Angle**
  `Used by [User Type] to [Result].`

## Social Proof

- `Trusted by [Number]+ [Audience Type]`
- `"Zenith helped us [Outcome] in [Timeframe]."`
- `Before Zenith: [Old Friction]. After Zenith: [Result].`

## Pricing

- `Best for [User Type]`
- `Includes [Top 3 Wins]`
- `Start with [Plan Name], upgrade when [Trigger]`

## FAQ

Use FAQ to remove objections:

- Is this hard to customize?
- Can I launch without design experience?
- Does this work on mobile?
- How fast can I launch?
- Can I adapt this to my niche?
