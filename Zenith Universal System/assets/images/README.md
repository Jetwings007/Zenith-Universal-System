# Zenith Image Assets

Store optimized screenshots and demo assets here.

## Recommended Rules

- Prefer modern formats (`.webp`, optimized `.png`) for screenshots.
- Keep hero images below 250 KB when possible.
- Use descriptive file names:
  - `zenith-home-desktop.webp`
  - `zenith-home-mobile.webp`
  - `zenith-ai-variant.webp`
